# Sincronizar Diretórios com Rsync

Este repositório contém um script em Shell para sincronizar os arquivos de dois diretórios utilizando o comando `rsync`. O script oferece um menu de opções para facilitar a interação do usuário e sincroniza apenas os arquivos que foram modificados.

## Funcionalidades

1. **Sincronizar Diretórios**: Solicita ao usuário o caminho do diretório de origem (produção) e do diretório de destino (backup). Sincroniza os arquivos do diretório de origem para o diretório de destino, copiando apenas os arquivos que foram modificados.

2. **Sair**: Encerra o script.
